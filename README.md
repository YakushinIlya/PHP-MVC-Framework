# PHP MVC framework v1.1
Минимальная версия mvc фреймворка на чистом php с возможностью масштабирования.\
Добавить возможно любой функционал, все довольно наглядно и просто, основа уже есть.

## Что умеет фреймворк
На данный момент фреймворк может принимать маршруты, обрабатывать данные в контроллере и выводить на экран html шаблон с данными.
- Route - задаются маршруты в файле route/web.php подобным образом: ``` '/about' => ['GET', AboutController::class, 'index'] ``` или с параметрами которым нужно указать их тип (:int - число, :string - строка, :slug - только латинские символы, цифры и дефис) ``` '/users/{id:int}' => ['GET', UserController::class, 'show'] ```.
- Controller - контроллеры можно создавать в директории app/Http, например, ``` HomeController ``` с наследованием от ``` Controller ``` находящегося в той же директории. Возврат шаблона происходит таким образом: ``` $this->view() ```, в который передаются 2 параметра.
  1. Название шаблона.
  2. Передаваемые параметры.
- Views - представления находятся в директории views. Они представляют собой обычный php файл содержащий в себе html контент и принимающий переданные параметры как обычные php переменные.
- DB - класс со статическими методами для взаимодействия с базой данных. Уже работает на получение данных, добавление, изменение и удаление. Например: ``` DB::query("select * from users")->execute()->firstArray(); ```.

Настройки для подключения к базе данных находятся в файле .env в корневом каталоге.
